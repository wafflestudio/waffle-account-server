== Health API
서버의 상태를 확인할 수 있는 API입니다. 만약 200이 반환되지 않는다면 제보를 부탁드립니다.

include::health.adoc[]

== Public User API
앱, 웹 등 모든 곳에서 호출할 수 있는 API입니다. 회원가입, 로그인, 회원탈퇴, 자신의 정보 조회 등 사용자의 정보에 관한 것 담당합니다.

include::public_user.adoc[]

== Private User API
웹 백엔드 등 WaffleStudio VPC 내부에서만 호출할 수 있는 API입니다. 모든 사용자의 정보를 제한 조건 없이 조회하고 변경할 수 있습니다.

include::private_user.adoc[]

== Public Token API
앱, 웹 등 모든 곳에서 호출할 수 있는 API입니다. 사용자의 토큰을 관리합니다.

include::public_token.adoc[]

== Public Verification API
앱, 웹 등 모든 곳에서 호출할 수 있는 API입니다. 휴대폰 및 이메일 인증을 담당하는 API입니다.

include::public_verification.adoc[]
